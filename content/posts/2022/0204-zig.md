# zig やってみる

新しいめの言語 [zig](https://ziglang.org/) をやってみる。

少しさわってみたとろ以下のような感想。

|      | 言語仕様 | GC   |
|------|----------|------|
| zig  | 小       | no   |
| c    | 小       | no   |
| go   | 小       | 有り |
| rust | 大       | no   |
| d    | 大       | 有り |
| c++  | 特大     | no   |

`go` が `C` の後継ポジションかと思っていたが `zig` かもしれぬ。
`rust` は身につかなかったので、今年は `zig` やってみよう。

https://github.com/ousttrue/zig_note で実験中。
