# QuickFixとLocationList周り

## setqflist と setloclist

```
# システムでひとつ
setqflist({list} [, {action}[, {what}]])

# Window毎
# nr(Window番号)が増えている
setloclist({nr}, {list} [, {action}[, {what}]])
```
