python で地理情報を扱う¶

python で地図画像を作ろうと思ったら思いのほか大変だった(まだできていない)のでメモ。

Windows であれば

https://www.lfd.uci.edu/~gohlke/pythonlibs/

の助けを借りるのがよさそう。

GDAL¶

gdal-config

version ?
